# Decode_cipher
Brute force substitution cipher code to find all available mappings

IMPORTANT: This works for substitution cipher only !!!
this may take a long time to run, the more words your cipher code has.
however, the less words you have, the more possible solutions there will be. Find a balance

A separate file needs to be added with your cipher text in the same folder as this code, and needs to be named cipher (in txt format). Happy decoding
